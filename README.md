## 项目练习
做一个简单员工管理系统

#### 项目功能:
  1. 员工登录
  2. 员工新增
  3. 部门新增
  4. 部门列表
  5. 员工列表
  6. 员工修改
  7. 员工权限(新增员工,新增部门,显示本部/跨部员工,设置权限)

### 技术选型
  - 前端
    1. zui 前端视觉框架 http://zui.sexy(禅道团除出品)
    2. 使用 angularJs 实现数据绑定及业务逻辑
  - 后端
    - 直接使用 TreeQL的restFull Json api,进行数据交互
